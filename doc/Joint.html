Class: Joint

Joint

new Joint()

This represents a joint between two bodies.
Properties:
Name Type Description
bodyId1 String The id of the first entity that will be attached to this joint
bodyId2 String The id of the second entity that will be attached to this joint
bodyPoint1 Object A point on the first entity where be attached to the second body. If no point is specified, the first body will be attached at its center point.
jointAttributes Object an object with any other properties that should be mixed into the box2d joint definition.
