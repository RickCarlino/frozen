Class: Box

Box

new Box()

This wraps the box2d world that contains bodies, shapes, and performs the physics calculations.
Properties:
Name Type Description
intervalRate Number The number of cycles per second expected in update calcuations
adaptive Boolean Whether or not to try to compensate calculations based on performance
bodiesMap Object A map of the bodies in the box2d world referenced by their given userData
fixturesMap Object A map of the fixtures in the box2d world referenced by their given userData
jointsMap Object A map of the joints in the box2d world referenced by their given userData
b2World Object The instance of the Box2D.Dynamics.b2World world class that the bodies, fixtures, and joints are used in.
gravityX Number The x component of the b2World's gravity in meters/second squared
gravityY Number The y component of the b2World's gravity in meters/second squared
allowSleep Boolean Allow box2d to skip physics calculations on bodies at rest for performance
resolveCollisions Boolean Whether to add a listener to collision events. Default behavior adds collision data to entities on update cycle
contactListener Object A contact listener for callbacks on collision events. Default is this box itself.
scale Number The number of pixels that represnt one meter in the box2d world.
