Class: GameCore

GameCore

new GameCore()

The GameCore class provides the base to build games on.
Properties:
Name Type Description
statics Object A map of static constants for internal use
isRunning Boolean Whether or not the game should be running its loop
canvasId String The id of the canvas element to use render the game on
maxStep Number Max number of milliseconds between updates. (in case user switches tabs and requestAnimationFrame pauses)
contextType String The type of context to request from the canvas. 2d or 3d
height Number The height of the Game and canvas
width Number The width of the Game and canvas
resourceManager ResourceManager The ResourceManager to be used for game
inputManager InputManager The InputManager to be used for game
loadingForeground String The style to be used for the foreground while game resources are loading
loadingBackground String The style to be used for the background while game resources are loading
gameAreaId String The id of a DOM element that contains the game's canvas
canvasPercentage Number The percentage (0 to 1.0) of the height and width the canvas should use to fill in its container DOM element
Source:

Example
var myGame = new GameCore({
  canvasId: 'myCanvas',
  update: function(millis){
    // do updating of game state
  },
  draw: function(context){
    // do drawing of the game
  }
});

//start the game
myGame.run();
